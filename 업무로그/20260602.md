## 사장님 컨펌 대기

### Supabase 마이그레이션 직접 실행 필요
아래 SQL을 Supabase 대시보드 > SQL Editor에서 직접 실행해야 반영됩니다.

```sql
ALTER TABLE profiles ADD COLUMN IF NOT EXISTS is_mvp boolean NOT NULL DEFAULT false;

UPDATE profiles
SET
  is_mvp        = true,
  plan          = 'pro',
  plan_expires_at = NULL
WHERE is_mvp = false;
```

→ 실행하면 기존 가입자 전원 Pro + MVP 처리 완료
