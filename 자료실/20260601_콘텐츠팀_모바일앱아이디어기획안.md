# 뀨스튜디오 모바일 앱 아이디어 기획안

작성일: 2026-06-01
작성팀: 콘텐츠팀
미션: 콘텐츠/퍼스널 브랜딩 관점에서 모바일 앱으로 만들면 효과적인 아이디어 3가지

---

## 아이디어 1. 뀨루틴 (KkuRoutine)

**한 줄 요약**
1인 창업자·크리에이터를 위한 하루 루틴 체크 앱. "오늘 나다운 하루를 보냈는가"를 매일 묻는다.

**왜 모바일이어야 하는가**
루틴 관리는 아침·저녁 특정 시간에 행동을 유도해야 효과가 있다. 푸시 알림이 없으면 사용자는 까먹는다. 오프라인 저장으로 인터넷 없이도 기록이 가능해야 하고, 잠금화면 위젯으로 오늘의 달성률을 항상 노출할 수 있다. 웹에서는 이 두 가지가 불가능하다.

**콘텐츠 / 수익 구조**
- 콘텐츠: 사장님 인스타(@kku._.ui)에 "내 루틴 앱으로 한 달 살아보기" 시리즈 → 앱 자연 노출
- 기본 기능 무료, 루틴 템플릿 팩(개발자용/크리에이터용/뷰티 루틴) 인앱 구매 (1,900원~3,900원)
- 뀨 템플릿샵과 연계: 앱에서 사용하는 루틴 카드 디자인 = 템플릿샵 굿즈로도 판매
- 월 구독(뀨루틴 PRO): 통계, 루틴 공유 기능, 배경 테마 → 2,900원/월

**예상 제작 기간**
- MVP(루틴 등록 + 알림 + 달성 체크): 3~4주
- PRO 기능 추가: 이후 2~3주

## 아이디어 2. 뀨로그 (KkuLog)

**한 줄 요약**
인스타·티스토리 운영자를 위한 콘텐츠 발행 기록 앱. "오늘 올린 거 있어?" 알림 하나로 꾸준함을 만든다.

**왜 모바일이어야 하는가**
콘텐츠 발행의 가장 큰 적은 "오늘도 못 올렸네"가 쌓이는 것이다. 푸시 알림으로 발행 요일을 지켜주고, 카메라 촬영 후 바로 기록하는 흐름이 모바일에서만 자연스럽다. 오프라인 초안 저장 기능으로 아이디어가 떠오른 순간 즉시 메모할 수 있다. 또한 앱에서 바로 인스타 공유 시트를 호출할 수 있다.

**콘텐츠 / 수익 구조**
- 콘텐츠: "100일 콘텐츠 챌린지" 기획 → 앱 내 챌린지 참여 + 인스타 해시태그 캠페인으로 바이럴
- 티스토리 연동: 포스팅 제목·날짜 기록 → 통계 대시보드 제공
- 수익: 무료 기본 / 연간 구독 9,900원(연간 통계, CSV 내보내기, 잠금화면 위젯)
- 사장님 브랜딩: "뀨로그로 200일 연속 발행한 방법" 티스토리 포스팅 → SEO 유입

**예상 제작 기간**
- MVP(발행 기록 + 알림 + 스트릭): 2~3주
- 플랫폼 연동/통계: 이후 3~4주

## 아이디어 3. 뀨무드 (KkuMood)

**한 줄 요약**
뷰티·라이프스타일 크리에이터를 위한 감성 무드 다이어리 앱. 오늘의 컬러와 한 줄로 하루를 기록한다.

**왜 모바일이어야 하는가**
무드 기록은 감정이 가장 생생한 순간, 즉 이동 중·자기 전에 이루어진다. 잠금화면 위젯으로 오늘의 무드 컬러를 띄워 앱 자체가 인테리어가 된다. 사진 첨부(카메라/갤러리 직접 접근)와 위젯 커스터마이징은 웹에서 구현할 수 없다. 또한 매일 밤 "오늘 무드는?" 푸시 알림이 습관 형성의 핵심이다.

**콘텐츠 / 수익 구조**
- 콘텐츠: 앱에서 생성된 무드 카드를 인스타 스토리에 바로 공유 → 자연 바이럴. 사장님이 직접 매일 무드 카드 공유하며 사용자 사례 축적
- 수익: 무료 기본 테마 / 시즌 테마 팩(봄·여름·가을·겨울) 인앱 구매 각 1,900원 / 연간 PRO 12,000원(무제한 테마, 연간 리포트, 잠금화면 위젯 전체 해제)
- 뀨 레터 연계: 월간 뀨 레터에 "이달의 무드 컬러 분석" 코너 → 앱 데이터 기반 콘텐츠로 활용
- 부가 수익: 무드 컬러 기반 굿즈(스티커, 엽서) 마플샵 연계 판매

**예상 제작 기간**
- MVP(무드 선택 + 한 줄 기록 + 위젯 + 알림): 3~4주
- 테마 팩 상점 + 공유 카드 생성: 이후 2~3주
