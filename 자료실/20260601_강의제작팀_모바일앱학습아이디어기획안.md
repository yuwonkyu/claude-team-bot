## 아이디어 1. DailyPrompt

**한 줄 요약**
하루 1개 AI 프롬프트를 실습하고, 결과를 기록하며 프롬프트 실력을 쌓는 마이크로러닝 앱

**왜 모바일 앱이어야 하는가**
- 프롬프트 학습은 "오늘 하나만 해보자"는 습관에 최적화된 형태가 필요하다
- 출퇴근 중 5분, 점심 후 5분처럼 틈새 시간에 소비되는 콘텐츠다
- 푸시 알림("오늘의 프롬프트가 도착했습니다")으로 리텐션을 유지할 수 있다
- 결과를 앱 내에 스크린샷으로 저장하고 공유하는 행동이 자연스럽게 이루어진다

**학습 콘텐츠 형태**
- 매일 1개 프롬프트 카드 공개 (카테고리: 업무/창작/마케팅/코딩 보조)
- 프롬프트 → 실습 → 내 결과 기록 → 모범 예시 확인의 3단계 루프
- 30일 챌린지 형태로 묶어 완주 뱃지 제공
- 누적 실습 횟수, 연속 출석 스트릭 표시

**수익 모델**
- 무료: 하루 1개 프롬프트 열람
- 구독 (월 4,900원): 전체 프롬프트 라이브러리 무제한 + 카테고리 필터 + 결과 히스토리
- 연간 구독 (39,000원): 월간 대비 약 33% 할인

**예상 제작 기간**
- 앱 MVP: 6주 (React Native/Expo, 콘텐츠 DB는 Supabase 또는 Airtable)
- 초기 콘텐츠 (30일치 프롬프트 카드): 별도 2주
- 총 8주

## 아이디어 2. SnippetDeck

**한 줄 요약**
개발자가 실무에서 바로 쓰는 코드 스니펫을 플래시카드 방식으로 외우고, 실제로 타이핑 실습하는 앱

**왜 모바일 앱이어야 하는가**
- 코드 암기는 반복 노출이 핵심이다. 이동 중 5분, 자기 전 10분의 플래시카드 학습이 가장 효과적이다
- 오프라인 저장을 지원하면 지하철·비행기 안에서도 학습이 가능하다
- 타이핑 실습을 모바일 키보드로 진행하면 "실제로 손가락이 기억한다"는 학습 효과가 있다
- Spaced Repetition(간격 반복) 알고리즘을 탑재하면 앱이 알아서 복습 시점을 알려준다

**학습 콘텐츠 형태**
- 언어/프레임워크별 스니펫 덱 (React, Python, SQL, Expo, Tailwind 등)
- 카드 앞면: 상황 설명 ("배열에서 중복 제거하려면?")
- 카드 뒷면: 코드 확인 + 직접 타이핑 채점
- 틀린 카드는 다음 세션에 자동 재등장 (SRS 방식)
- 사용자가 자신만의 스니펫 덱을 추가하는 기능

**수익 모델**
- 무료: 기본 덱 3개 (React 필수 훅, Python 기초, SQL 기초)
- 구독 (월 6,900원): 전체 덱 20개 이상 + 커스텀 덱 무제한 생성 + 오프라인 모드
- 연간 구독 (59,000원): 월간 대비 약 29% 할인
- 팀 플랜 (월 29,000원 / 최대 5인): 팀 공유 덱 생성 가능

**예상 제작 기간**
- 앱 MVP: 7주 (SRS 로직 구현 포함)
- 초기 콘텐츠 (덱 5개, 카드 200장): 별도 2주
- 총 9주

## 아이디어 3. LaunchLog

**한 줄 요약**
1인 창업자가 사이드 프로젝트를 단계별로 런칭하도록 안내하는 체크리스트 + 학습 가이드 앱

**왜 모바일 앱이어야 하는가**
- 1인 창업자는 책상에 앉아 공부하는 사람이 아니다. 이동 중, 카페에서, 자기 전에 "오늘 뭘 해야 하지?"를 확인한다
- 푸시 알림으로 "오늘의 런칭 미션"을 전달하면 중단 없이 프로젝트를 진행하게 된다
- 오프라인 체크리스트는 인터넷 없이도 작동해야 하는 상황(이동 중 기획)이 많다
- 내 프로젝트 진행률이 한눈에 보이는 대시보드가 동기부여를 유지시킨다

**학습 콘텐츠 형태**
- "아이디어 → 검증 → MVP → 런칭 → 첫 수익" 5단계 로드맵 구조
- 각 단계마다 체크리스트 항목 + 2~3분짜리 마이크로 가이드 텍스트 또는 음성 제공
- 실제 사례 (뀨스튜디오, instalink 등) 기반의 실전 예시 포함
- 체크 완료 시 뱃지와 진행률 업데이트로 성취감 제공
- 프로젝트별로 별도 보드를 만들어 여러 사이드 프로젝트 병행 관리 가능

**수익 모델**
- 무료: 프로젝트 1개, 1단계(아이디어~검증)까지
- 구독 (월 7,900원): 프로젝트 무제한 + 전체 5단계 + 음성 가이드 + 오프라인 모드
- 연간 구독 (69,000원): 월간 대비 약 27% 할인
- 원타임 구매 옵션 (99,000원): 구독 없이 영구 이용 (LTV 높은 유저 유인)

**예상 제작 기간**
- 앱 MVP: 6주 (프로젝트 관리 구조 + 콘텐츠 뷰어)
- 초기 콘텐츠 (5단계 로드맵, 체크리스트 100항목, 가이드 30편): 별도 3주
- 총 9주

## 강의제작팀 의견

세 아이디어 모두 사장님의 기존 역량(React Native/Expo, AI 강의 콘텐츠)과 직접 연결됩니다.

우선순위 추천 순서:

1. **LaunchLog** - 사장님 본인의 창업 경험이 가장 강력한 콘텐츠 자산이 되며, 뀨스튜디오·instalink 사례를 그대로 강의 소재로 활용 가능. 타깃이 명확하고 재구독 이유가 분명합니다.
2. **DailyPrompt** - 시장이 넓고 진입 장벽이 낮습니다. Claude API 강의와 연계해 교차 프로모션이 가능합니다.
3. **SnippetDeck** - 개발자 타깃으로 니치하지만 충성도가 높습니다. 사장님이 직접 개발자이므로 콘텐츠 품질 보증이 됩니다.

컨펌 요청 사항 없음 (기획안 단계). 진행 방향 결정 후 상세 커리큘럼 또는 앱 설계서 작업을 이어서 진행할 수 있습니다.
