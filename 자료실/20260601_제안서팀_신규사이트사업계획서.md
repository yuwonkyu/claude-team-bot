## 후보 A: AI 툴 큐레이션 디렉토리

### 1. 왜 해야 하는가 (시장 근거)

**한국 시장 수요 근거**

국내에서 AI 도구를 처음 접하는 사용자들은 "어떤 AI를 쓰면 되나요?"라는 질문을 반복한다. 네이버 블로그·유튜브에서 "AI 툴 추천" 키워드는 2024~2025년 사이 검색량이 급격히 증가했으며, 이는 도구 선택의 피로감이 한국어 큐레이션 수요로 직결되고 있음을 의미한다.

- 글로벌 AI 도구 수는 2025년 기준 수만 개 이상
- 한국어 설명·리뷰가 붙은 디렉토리 수요는 여전히 공급 부족
- 애드센스 승인 후 SEO 트래픽으로 수익화가 가능한 대표적 구조

**수요 키워드 (한국어)**
> "AI 툴 추천", "무료 AI 사이트 모음", "ChatGPT 대체 AI", "이미지 생성 AI 비교", "직장인 AI 도구"

### 2. 경쟁사 현황

| 서비스명 | 특징 | 약점 |
|----------|------|------|
| AI판 (aipan.kr) | 영상·이미지·텍스트 분야별 분류, 뉴스 포함 | UI 노후, 설명 얕음 |
| AI모아 (ai모아.com) | 2,338개 툴 등록, 업무별 필터 | 디자인 밀도 낮음, SEO 약함 |
| 다모아AI (damoa.ai) | 분야별 검색 가능 | 갱신 주기 불분명 |
| Toolify (toolify.ai/ko) | 글로벌 2만여 개, 한국어 지원 | 한국 특화 콘텐츠 없음 |

**결론:** 경쟁자는 이미 여럿 존재한다. 단순 나열 방식의 디렉토리는 차별화가 어렵다.

### 3. 차별화 포인트

경쟁에서 살아남으려면 단순 목록이 아닌 **"직군별·용도별 실사용 큐레이션"** 으로 포지셔닝해야 한다.

- **직군 타겟팅:** "마케터용 AI 5종 세트", "개발자용 AI 루틴" 등 역할 기반 묶음
- **한국어 실사용 후기 중심:** 영어 설명 번역이 아닌 직접 써본 뉘앙스
- **instalink 연동:** 바이오링크 사용자들에게 AI 툴 추천 → 자연스러운 크로스 트래픽
- **SNS 연결:** @kku._.ui 인스타에서 "이달의 AI 툴" 콘텐츠 → 사이트 유입 루트

### 4. 수익 모델

| 수익원 | 방식 | 예상 시점 | 예상 수익 |
|--------|------|----------|----------|
| 구글 애드센스 | 트래픽 기반 광고 | 승인 후 즉시 | 초기 월 1~5만원 → 성장 시 월 20~50만원 |
| 제휴 링크 (어필리에이트) | 유료 AI 툴 추천 링크 | 바로 가능 | 전환 1건당 수천~수만원 |
| 스폰서 등록 | AI 스타트업이 자사 툴 상단 노출 비용 지불 | 트래픽 확보 후 | 월 10~30만원/건 |

> 애드센스 수익은 월 방문자 1만 명 기준 약 5~15만원 수준. 트래픽 확보가 선행 조건이다.

### 5. 제작 기간 (1인 개발 기준)

| 단계 | 기간 | 작업 내용 |
|------|------|----------|
| 1단계: MVP | 2~3주 | Next.js + 정적 데이터로 100개 툴 등록, 카테고리 분류 |
| 2단계: 검색/필터 | 1주 | 키워드 검색, 직군별 필터 추가 |
| 3단계: SEO | 지속 | 각 툴 개별 페이지, 메타태그 최적화 |
| 4단계: 수익화 | MVP 이후 | 애드센스 신청, 제휴 링크 삽입 |

**총 예상 기간:** MVP 기준 3~4주

### 6. 6월 내 실현 가능성 평가

| 항목 | 평가 | 비고 |
|------|------|------|
| 개발 난이도 | 낮음 | 정적 사이트 수준, DB 없이 JSON으로 시작 가능 |
| 6월 MVP 출시 | 가능 | 3~4주면 100개 툴 기준 MVP 충분 |
| 수익 발생 시점 | 느림 | 애드센스는 트래픽 쌓이는 데 3~6개월 필요 |
| 차별화 어려움 | 높음 | 이미 경쟁자 다수 |

**실현 가능성: 중간** — 만들기는 쉽지만, "왜 여기서 봐야 하나"를 설득하는 콘텐츠 전략이 없으면 트래픽이 모이지 않는다.

### 7. 리스크와 해결 방안

| 리스크 | 해결 방안 |
|--------|----------|
| 경쟁 포화 (AI판, AI모아 등 선점) | 직군 특화 + 사장님 실사용 경험 기반 큐레이션으로 차별화 |
| SEO 성과 느림 | 티스토리 블로그와 연동, 상위 노출 가능한 롱테일 키워드 공략 |
| 유지보수 부담 | 주 1회 업데이트 루틴 설계, AI로 초안 자동화 |
| 제로클릭 검색 증가로 트래픽 감소 | 제휴 링크·스폰서 수익으로 애드센스 의존도 분산 |

### 8. 추천 여부

**조건부 추천 (후순위)**

경쟁이 이미 형성된 시장이다. 단순 디렉토리로는 차별화가 어렵고, 트래픽 확보까지 시간이 걸린다. 단, 후보 B 또는 C를 메인으로 운영하면서 콘텐츠 SEO 보조 사이트로 운영하거나, instalink와 번들로 AI 툴 추천 섹션을 붙이는 방식이 더 효율적일 수 있다.

## 후보 B: 한국어 프롬프트 생성기

### 1. 왜 해야 하는가 (시장 근거)

**한국 시장 수요 근거**

"ChatGPT한테 어떻게 물어봐야 해요?"는 2024~2026년 가장 많이 검색되는 AI 관련 질문 중 하나다. 한국어 사용자는 영어 기반 프롬프트 가이드를 그대로 사용하기 어렵고, 맥락·경어·직장 문화에 맞는 한국형 프롬프트 생성에 특화된 서비스가 부족하다.

- "프롬프트 작성법" 네이버 블로그 조회수 건당 수만~수십만
- AI 사용 초보자(비개발자) 비율이 국내 AI 사용자의 80% 이상으로 추정
- 직장인 대상 ChatGPT 활용 강의가 오프라인·온라인 전방에서 폭발적으로 성장 중

### 2. 경쟁사 현황

| 서비스명 | 특징 | 약점 |
|----------|------|------|
| Promfit (promfit.ai.kr) | AI 툴별 자동 프롬프트 생성 | UI 단순, 직군 특화 없음 |
| Pocket Prompt (pocket-prompt.com) | 프롬프트 저장·관리 크롬 확장 중심 | 생성 기능 약함, 웹 단독 약함 |
| OpenPrompt (prpt.ai) | 프롬프트 공유 커뮤니티 | 생성 기능 없음, 탐색형 |
| 프롬프트허브 (prompts.ai.kr) | 텍스트·이미지 프롬프트 공유 | 공유 중심, 생성 기능 부재 |
| Prompti (prompti.kr) | 프롬프트 마켓플레이스 | 구매 모델, 직접 생성 아님 |

**결론:** "직접 입력 → 한국어 최적화 프롬프트 자동 생성" 기능에 특화된 서비스는 상대적으로 부족하다.

### 3. 차별화 포인트

- **직군별 맥락 인식:** "마케터", "개발자", "HR 담당자", "자영업자" 등 역할을 선택하면 해당 직군에 최적화된 프롬프트 생성
- **경어·어투 설정:** 보고서용/캐주얼용/공식 이메일용 등 한국어 뉘앙스 선택
- **목적 카테고리 분류:** 요약 / 번역 / 글쓰기 / 분석 / 아이디어 / 코딩 등
- **프리미엄 차별화:** 무료: 기본 생성 / 유료: 고급 프롬프트 템플릿, 저장 기능, 무제한 생성
- **instalink 연동 가능성:** 바이오링크 사용자에게 "AI 프롬프트 생성기" 탭 연결 → 크로스 트래픽

### 4. 수익 모델

| 수익원 | 방식 | 예상 시점 | 예상 수익 |
|--------|------|----------|----------|
| 구글 애드센스 | 무료 사용자 대상 광고 | 승인 후 즉시 | 월 5~20만원 (트래픽 확보 후) |
| 프리미엄 플랜 | 월 구독 3,900원~9,900원 | 출시 후 즉시 판매 가능 | 구독자 50명 = 월 20~50만원 |
| API 연동 비용 | Claude/GPT API 사용량 기반 수익 분배 | 유료 플랜 연동 시 | 플랜 가격에 포함 |

> 프리미엄 구독 모델은 instalink SaaS 운영 경험이 있으므로 구조적으로 이미 검증됨.

### 5. 제작 기간 (1인 개발 기준)

| 단계 | 기간 | 작업 내용 |
|------|------|----------|
| 1단계: MVP | 2~3주 | 직군 선택 + 목적 선택 + 프롬프트 생성(Claude API 연동) |
| 2단계: 저장·공유 | 1주 | 생성된 프롬프트 저장, 클립보드 복사 기능 |
| 3단계: 프리미엄 | 1주 | 결제 연동 (Stripe 또는 토스페이먼츠), 구독 관리 |
| 4단계: SEO + SNS | 지속 | 직군별 랜딩페이지, @kku._.ui 콘텐츠 연동 |

**총 예상 기간:** MVP 기준 3주, 유료 플랜 포함 4~5주

### 6. 6월 내 실현 가능성 평가

| 항목 | 평가 | 비고 |
|------|------|------|
| 개발 난이도 | 중간 | Claude/GPT API 연동 필요, Next.js로 충분히 가능 |
| 6월 MVP 출시 | 가능 | 무료 기능 기준 3주 내 가능 |
| 수익 발생 시점 | 빠름 | 유료 구독 플랜은 출시 즉시 판매 가능 |
| 차별화 강도 | 중간~높음 | 직군·어투 조합은 국내 미개척 |

**실현 가능성: 높음** — 기술 난이도가 현재 기술스택 범위 내이며, 수익화 경로가 명확하다.

### 7. 리스크와 해결 방안

| 리스크 | 해결 방안 |
|--------|----------|
| Claude/GPT API 비용 | 무료 플랜에 월 생성 횟수 제한(예: 10회) 설정, 유료만 무제한 |
| 경쟁자 빠른 진입 | 직군 특화 UX는 단기간 모방 어려움 → 빠른 출시로 선점 |
| 사용자 재방문 유도 어려움 | 저장·히스토리 기능 + 이메일 뉴스레터로 재방문 유도 |
| AI 정확도에 대한 불만 | 생성 후 직접 수정 가능한 에디터 제공 |

### 8. 추천 여부

**강력 추천 (1순위 후보)**

한국어 사용자의 실제 페인포인트(프롬프트 작성 어려움)를 직접 해결하며, 기존 instalink SaaS 운영 경험과 구조적으로 동일하다. Claude API를 이미 활용 중이므로 기술 학습 비용이 낮고, 프리미엄 플랜으로 빠른 수익 전환이 가능하다. 6월 내 MVP 출시가 현실적이다.

## 후보 C: GPT 프롬프트 템플릿 판매 사이트

### 1. 왜 해야 하는가 (시장 근거)

**한국 시장 수요 근거**

"검증된 프롬프트를 사고 싶다"는 수요는 분명히 존재한다. 글로벌 마켓플레이스 PromptBase는 프롬프트를 $1.99~$9.99에 판매하며 시장을 형성했고, 국내에서도 GPTsKOREA, Prompti.kr 등이 유사 모델로 운영 중이다. 특히 "직군별 업무 자동화 프롬프트 묶음"은 직장인 수요가 확실하다.

- 국내 ChatGPT 사용자 중 업무 목적 비율: 추정 60% 이상
- 프롬프트 템플릿 구매 의향: AI 활용 고급 사용자 중 높음
- 콘텐츠크리에이터·마케터·HR 담당자 등 직군별 프롬프트 묶음 수요 존재

### 2. 경쟁사 현황

| 서비스명 | 특징 | 약점 |
|----------|------|------|
| Prompti (prompti.kr) | 국내 프롬프트 마켓, 판매 가능 | 규모 작음, 인지도 낮음 |
| GPTsKOREA | GPT 소스 키트 판매 (1~3만원) | 비개발자 접근성 낮음 |
| PromptBase (글로벌) | 글로벌 마켓, 영어 중심 | 한국어 콘텐츠 부족 |
| 크몽·탈잉 | 강의+템플릿 번들 판매 | 플랫폼 수수료 높음 (20~30%) |
| 오픈프롬프트 | 무료 공유 중심 | 유료화 구조 없음 |

**결론:** 한국어 직군 특화 프롬프트 템플릿을 체계적으로 판매하는 독립 사이트는 아직 공백이다.

### 3. 차별화 포인트

- **직군별 패키지:** 마케터용 30종, HR용 20종, 스타트업 대표용 25종 등 묶음 판매
- **즉시 복사·적용 UX:** 구매 후 바로 복사해서 ChatGPT/Claude에 붙여넣기
- **업데이트 보장:** "AI 업데이트 시 프롬프트도 업데이트" — 구독 모델 명분
- **독립 사이트의 신뢰도:** 크몽·탈잉 수수료 없이 직접 판매 → 가격 경쟁력
- **사장님 포지셔닝:** 1인 개발자 + AI 실사용자라는 배경이 "써본 사람이 만든 템플릿"으로 설득력 강함

### 4. 수익 모델

| 수익원 | 방식 | 예상 시점 | 예상 수익 |
|--------|------|----------|----------|
| 단건 판매 | 직군별 패키지 9,900원~29,000원 | 출시 즉시 | 월 10건 = 10~30만원 |
| 구독 모델 | 월 7,900원, 전 카테고리 접근 | 출시 즉시 | 구독 30명 = 월 24만원 |
| 번들 할인 | 3~5개 패키지 묶음 할인 | 출시 즉시 | 객단가 상승 |
| 기업 라이선스 | 팀 단위 구매 (협의 가격) | 입소문 후 | 건당 10만원 이상 |

> 디지털 상품 특성상 제작 후 판매 비용이 거의 0원. 초기 수익률이 가장 높은 모델이다.

### 5. 제작 기간 (1인 개발 기준)

| 단계 | 기간 | 작업 내용 |
|------|------|----------|
| 1단계: 콘텐츠 제작 | 1~2주 | 직군 3~5개, 각 20~30종 프롬프트 제작 |
| 2단계: 사이트 개발 | 1~2주 | 상품 목록, 상세 페이지, 결제 연동 (토스페이먼츠) |
| 3단계: 마케팅 | 출시 직후 | @kku._.ui 인스타, 티스토리 SEO 포스팅 |
| 4단계: 구독 시스템 | 선택 | 단건 판매로 먼저 시작 후 구독 추가 가능 |

**총 예상 기간:** MVP (단건 판매) 기준 2~3주

### 6. 6월 내 실현 가능성 평가

| 항목 | 평가 | 비고 |
|------|------|------|
| 개발 난이도 | 낮음~중간 | 결제 연동만 해결되면 나머지는 정적 사이트 수준 |
| 6월 MVP 출시 | 충분히 가능 | 2~3주면 단건 판매 구조 완성 |
| 수익 발생 시점 | 빠름 | 첫 판매부터 수익 발생 (수수료 0) |
| 콘텐츠 제작 부담 | 있음 | 프롬프트 템플릿 직접 작성 필요 (AI 활용 가능) |

**실현 가능성: 매우 높음** — 개발 복잡도가 가장 낮고, 수익 전환이 가장 직접적이다.

### 7. 리스크와 해결 방안

| 리스크 | 해결 방안 |
|--------|----------|
| 콘텐츠 품질 검증 | 사장님이 실제 업무에 사용한 프롬프트 위주로 구성 → 신뢰도 확보 |
| 복사·무단 공유 | 완전 방지는 어렵지만, 구독 모델로 정기 업데이트 가치 제공 |
| 초기 트래픽 없음 | 인스타 @kku._.ui + 티스토리 SEO로 초기 유입 확보 |
| ChatGPT 자체 업데이트로 템플릿 무력화 | "AI 업데이트 반영" 구독 명분으로 역이용 |
| 저가 경쟁 | 크몽 등 플랫폼 없이 직접 판매 → 수수료 없어 가격 경쟁력 확보 |

### 8. 추천 여부

**강력 추천 (공동 1순위 또는 첫 번째 출시 후보)**

개발 복잡도가 가장 낮고, 수익 전환이 가장 빠르다. 콘텐츠(프롬프트 템플릿) 제작이 핵심 작업이며, Claude AI를 활용해 빠르게 만들 수 있다. 기존 instalink SaaS 결제 시스템 경험이 있어 결제 연동 허들도 낮다. 단, 트래픽을 외부(SNS, 블로그)에 의존해야 하므로 마케팅 플랜이 필수다.
