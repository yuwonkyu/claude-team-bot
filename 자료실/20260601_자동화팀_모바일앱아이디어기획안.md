# 뀨스튜디오 모바일 앱 아이디어 기획안 (자동화팀)
작성일: 2026-06-01 | 담당: 자동화팀
미션: 1인 개발자(Expo/React Native)가 만들 수 있는 모바일 특화 앱 아이디어 3가지

---

## 아이디어 1: LinkPulse (인스타 바이오링크 관리 앱)

**앱명:** LinkPulse — instalink 모바일 파트너 앱

**한 줄 요약:** instalink 사용자가 스마트폰으로 링크 클릭수·유입 통계를 실시간 알림받고, 현장에서 즉시 링크를 추가/수정하는 모바일 관리 앱

**왜 모바일이어야 하는가:**
- 크리에이터는 PC 앞에 없는 경우가 대부분 — 촬영 현장, 팝업 행사, 이동 중
- "지금 막 영상 올렸는데 링크 빠르게 바꿔야 함" 같은 순간 대응은 모바일에서만 가능
- 실시간 클릭 폭등(바이럴 순간)을 푸시 알림으로 즉시 캐치해야 대응 가능
- QR코드를 카메라로 바로 스캔해서 자기 페이지 미리보기/공유 가능

**핵심 기능 (모바일 특화):**
- 푸시 알림: 클릭수 임계치 초과 시 알림 ("오늘 팔로워 500명이 클릭했습니다")
- 위젯: 홈화면 위젯으로 오늘 클릭수 실시간 표시 (iOS 위젯, Android 위젯)
- 카메라: QR코드 생성 + 즉시 카메라로 스캔 테스트
- 오프라인 캐시: 마지막 통계 오프라인 열람 가능
- 빠른 링크 추가: 공유 익스텐션으로 타 앱에서 URL 즉시 추가
- 잠금화면 위젯: 오늘 방문자 수 잠금화면에 표시

**수익 모델:**
- instalink Pro 구독 번들 (앱 단독 판매 없음, Pro 플랜 4,900원/월 가입 유도 도구)
- 앱 자체는 무료 → instalink Pro 전환율 상승 효과

**예상 제작 기간:** 4~5주 (instalink API 기존 활용, Expo 기반)

## 아이디어 2: DailyLog (1인 개발자 빌드 일지 앱)

**앱명:** DailyLog — 개발자 손글씨 회고 & 빌드 추적 앱

**한 줄 요약:** 하루 3분, 오늘 만든 것·막힌 것·내일 할 것을 기록하고 GitHub 커밋과 자동 연결해 주간 회고를 AI가 자동 생성해주는 앱

**왜 모바일이어야 하는가:**
- 회고는 PC 앞에 앉아 쓰는 것이 아니라 침대에서 자기 전 3분에 이루어짐
- 푸시 알림 없이는 매일 기록 습관 형성이 불가능 (웹 앱은 열어야 함)
- 음성 메모 기능: 이동 중 아이디어가 떠오를 때 마이크로 즉시 기록
- 위젯으로 오늘 커밋 수 + 기록 여부를 홈화면에서 확인 → 습관 강화
- 앱 전용 스트릭(연속 기록) 시각화 → 게임화 요소

**핵심 기능 (모바일 특화):**
- 푸시 알림: 매일 밤 9시 "오늘 빌드 일지 쓸 시간입니다" 커스텀 알림
- 마이크/음성 입력: 음성 → 텍스트 자동 변환 기록 (Expo Speech)
- 위젯: 오늘 커밋수 + 기록 완료 여부 홈화면 위젯
- 오프라인 완전 지원: 네트워크 없어도 기록, 나중에 동기화
- GitHub 연동: 커밋 로그 자동 불러와 일지에 첨부
- AI 주간 회고 생성: 7일치 기록 → Claude API → 주간 요약 + 성장 포인트 추출
- 스트릭 카운터: 연속 기록 일수 시각화 (GitHub 잔디 스타일)

**수익 모델:**
- 무료: 최근 7일 기록, 기본 알림
- Pro 구독 월 4,900원: GitHub 연동, AI 주간 회고, 무제한 기록, 위젯
- 연간 결제 39,000원 (34% 할인)
- 앱스토어 + 구글플레이 인앱 결제 (수수료 15~30% 감안한 가격 설정)

**예상 제작 기간:** 5~6주 (Expo + Supabase + Claude API)

## 아이디어 3: SpotAlert (소상공인 실시간 리뷰 대응 앱)

**앱명:** SpotAlert — 네이버·카카오 리뷰 즉시 알림 & AI 답변 앱

**한 줄 요약:** 네이버 플레이스·카카오맵·구글맵에 새 리뷰가 달리면 1분 안에 푸시 알림, 앱에서 AI 답변 초안을 바로 확인하고 원클릭 복사해서 답변하는 소상공인 필수 앱

**왜 모바일이어야 하는가:**
- 소상공인은 PC 앞에 없음 — 가게에서 서서 스마트폰으로 모든 것을 처리
- 리뷰에 24시간 내 답변하면 알고리즘 노출 상승 → 빠른 알림이 곧 매출
- 악의적 리뷰의 경우 빠른 대응이 브랜드 이미지 손상 방지에 결정적
- 위치 기반: 가게 반경 내 진입 시 "오늘 리뷰 확인하세요" 위치 알림 가능
- 카메라: 리뷰에 사진 첨부 답변할 때 즉석 촬영 후 바로 업로드

**핵심 기능 (모바일 특화):**
- 실시간 푸시 알림: 새 리뷰 등록 즉시 알림 (네이버 플레이스 크롤링 + 웹훅)
- AI 답변 초안: 리뷰 내용 분석 → Claude API → 감사/사과/해명 맞춤 초안 생성
- 원클릭 복사: 초안 복사 → 플랫폼 앱으로 이동 → 붙여넣기
- 오프라인 큐: 네트워크 없을 때 알림 저장, 연결 시 동기화
- 위치 알림: 가게 반경 진입 시 당일 미답변 리뷰 알림
- 리뷰 통계 위젯: 이번 주 평균 별점 + 미답변 수 홈화면 위젯
- 멀티 플레이스: 지점이 여러 개인 경우 한 앱에서 통합 관리
- 카메라 통합: 음식/인테리어 사진 즉석 촬영해서 답변에 활용

**수익 모델:**
- 무료 14일 체험
- 1개 플레이스: 월 9,900원
- 3개 플레이스: 월 19,900원
- 무제한 (프랜차이즈): 월 49,000원
- 연간 결제 시 2개월 무료

**예상 제작 기간:** 6~7주 (Expo + n8n 리뷰 크롤링 워크플로우 + Claude API)
- n8n이 백엔드 크롤링/알림 발송을 담당 → 1인 개발 부담 대폭 절감
- Expo Push Notification으로 iOS/Android 동시 지원

## 자동화팀 의견

**SpotAlert를 1순위로 추천하는 이유:**
- 기존 종합 보고서 "소상공인 AI 리뷰 응답 자동화" SaaS와 연계 — 웹 SaaS로 먼저 출시 후 앱으로 확장하는 경로가 자연스러움
- n8n 워크플로우로 크롤링/알림 파이프라인을 구축하면 앱은 UI만 담당 → 1인 개발 현실적
- 소상공인 시장은 앱스토어보다 지인 추천/입소문으로 확산 → 초기 마케팅 비용 낮음
- 모바일이 없으면 절대 불가능한 "가게에서 즉시 대응" 시나리오 → 웹 경쟁자 없음

**DailyLog를 2순위로 추천하는 이유:**
- 뀨스튜디오 사장님 본인이 직접 쓸 도구 → 개발 과정 자체가 콘텐츠
- RetroKit, DevRetro 아이템과 시너지 — 디지털 상품 라인업으로 연결 가능
- Expo의 음성입력, 오프라인, 위젯이 핵심 → 순수 웹으로 대체 불가
