## 아이디어 1. ReviewReply (소상공인 리뷰 즉시 응답 앱)

### 해결하는 문제
소상공인은 네이버 플레이스, 구글 맵, 카카오맵에 리뷰가 달려도 확인이 늦고, 답글 작성이 번거롭다. 리뷰 답글은 신뢰도와 SEO에 직결되지만 대부분 방치된다.

### 왜 모바일이어야 하는가
리뷰는 사장님이 가게에 있을 때, 즉 모바일로 확인한다. 새 리뷰 알림을 받는 즉시 AI 초안을 보고 한 탭으로 발행하는 경험은 웹에서는 불가능하다. 푸시 알림이 이 서비스의 핵심 트리거다.

### 핵심 기능
- 연동된 플랫폼(네이버, 구글)의 새 리뷰 감지 즉시 푸시 알림
- Claude API 기반 AI 답글 초안 자동 생성 (업종·톤 설정 가능)
- 초안 수정 후 한 탭 발행
- 별점별·키워드별 리뷰 통계 대시보드

### 수익 모델
- 무료: 월 10건
- 월 9,900원 (1개 매장, 무제한)
- 월 19,900원 (3개 매장, 브랜드 톤 커스터마이징)

### 예상 제작 기간
MVP 기준 5~6주. 리뷰 크롤링 또는 공식 API 연동 방식에 따라 변동. 네이버 플레이스 공식 API 미지원 시 웹뷰 기반 우회로 처리 가능.

## 아이디어 2. DailyShip (개발자 빌드 로그 & 스트릭 앱)

### 해결하는 문제
1인 개발자는 매일 무언가를 만들지만 기록하지 않는다. GitHub 잔디로는 "무엇을 만들었는지"가 남지 않는다. 동기 부여도 없고, 나중에 포트폴리오로 쓸 빌드 히스토리도 없다.

### 왜 모바일이어야 하는가
"오늘 뭐 만들었어?" 라는 질문에 대답하는 행위는 하루 마무리 시점, 즉 모바일로 이루어진다. 스트릭 유지 알림, 위젯, 잠금화면 빌드 현황 표시가 모바일에서만 가능하다. 커뮤니티 피드를 스크롤하며 다른 개발자의 빌드를 보는 경험도 모바일이 훨씬 자연스럽다.

### 핵심 기능
- 하루 1회 빌드 로그 기록 (텍스트 + 스크린샷 + URL)
- GitHub 연동으로 커밋 자동 요약 (AI)
- 연속 기록 스트릭 + 배지 시스템
- 공개 프로필 페이지 (포트폴리오로 활용 가능)
- iOS 위젯: 오늘 빌드 현황, 스트릭 카운트
- 커뮤니티 피드: 다른 개발자 빌드 탐색 및 응원

### 수익 모델
- 무료: 공개 프로필, 스트릭
- 월 4,900원 Pro: GitHub 자동 연동, AI 요약, 위젯, 비공개 로그
- 연간 39,000원

### 예상 제작 기간
MVP (로그 기록 + 스트릭 + 공개 프로필) 기준 6~7주. 커뮤니티 피드는 2차 업데이트로 분리 가능.

## 아이디어 3. CsReady (소상공인 고객 응대 자동화 앱)

### 해결하는 문제
인스타그램 DM, 카카오채널, 네이버 톡톡으로 같은 질문이 반복적으로 들어온다. "운영시간이 어떻게 되나요", "환불 되나요", "재입고 언제예요". 사장님은 이 반복 응대에 하루 1~2시간을 쓴다.

### 왜 모바일이어야 하는가
소상공인의 DM 응대는 99% 스마트폰으로 이루어진다. 알림이 오면 바로 답해야 한다. 앱에서 FAQ 세트를 미리 등록해두면, 새 메시지가 올 때 AI가 초안을 자동 생성해 알림으로 보내준다. "확인 후 탭 한 번으로 발송"하는 플로우가 이 서비스의 핵심이며, 이는 모바일 없이는 구현되지 않는 경험이다.

### 핵심 기능
- 카카오채널, 인스타 DM 연동 (Webhook 또는 공식 API)
- 업종별 FAQ 템플릿 라이브러리 (뷰티, 식음료, 핸드메이드 등)
- 수신 메시지 분류 + AI 답변 초안 즉시 생성 (푸시 알림 포함)
- 초안 수정 후 원클릭 발송
- 자주 묻는 질문 자동 학습 및 FAQ 추가 제안

### 수익 모델
- 무료: 월 30건, 1개 채널
- 월 12,900원: 무제한 + 3개 채널
- 월 24,900원: 무제한 + 전 채널 + 팀 계정 1명 추가

### 예상 제작 기간
MVP (카카오채널 단독 + FAQ 등록 + AI 초안) 기준 5~6주. 인스타 DM은 Graph API 심사가 필요하므로 2차 업데이트 권장.

## 고객응대팀 의견

세 아이디어 모두 React Native/Expo로 구현 가능하며 1인 운영 범위 안에 있습니다.

ReviewReply와 CsReady는 소상공인 타깃으로, 이미 종합보고서에서 1순위로 추천된 "소상공인 AI 리뷰 응답 자동화"의 모바일 확장판입니다. 웹 SaaS와 앱을 동시에 또는 순차적으로 가져가는 전략도 고려할 수 있습니다.

DailyShip은 개발자 커뮤니티 타깃으로, 뀨 레터(뉴스레터)와 연동하면 초기 사용자를 뀨스튜디오 팬베이스에서 빠르게 확보할 수 있습니다.
