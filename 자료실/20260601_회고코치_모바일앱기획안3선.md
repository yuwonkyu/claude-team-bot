## 아이디어 1. DayDiff

### 해결하는 문제
개발자·1인 창업자는 하루 단위의 생산성을 체감하지 못한 채 일주일이 지나간다.  
"오늘 뭘 했지?"를 밤에 떠올리려 하면 기억이 흐릿하다.  
회고를 하려 해도 기록이 없어서 감상만 남고, 패턴 파악이 안 된다.

### 왜 모바일이어야 하는가
- 행동이 일어나는 순간(이동 중, 커피 한 잔 마실 때, 코딩 세션 직후)에 기록할 수 있어야 한다.  
- 잠금화면 위젯으로 "오늘 체크인 했나?" 를 시각적으로 상기시킬 수 있다.  
- 푸시 알림으로 오후 6시 "오늘 가장 잘한 것 하나만 적어보세요" 를 보낼 수 있다.  
- 웹은 의식적으로 열어야 하지만, 모바일은 생활 속에 끼어든다.

### 핵심 기능
1. 30초 체크인: 오전·오후 각 1회, 3가지 질문(지금 하는 일 / 에너지 수준 1~5 / 한 줄 메모)  
2. 잠금화면 위젯: 오늘 체크인 횟수, 현재 에너지 평균, 연속 기록 스트릭  
3. 주간 자동 회고 리포트: 체크인 데이터 기반으로 "이번 주 에너지 피크 구간", "가장 자주 한 일 TOP3" 자동 생성  
4. 오프라인 우선: 인터넷 없어도 기록되고, 나중에 동기화  
5. 내보내기: 주간 리포트를 노션·슬랙·이메일로 원클릭 공유

### 수익 모델
- 무료: 체크인 기록 + 7일 히스토리  
- Pro 월 3,900원: 무제한 히스토리, 주간 리포트, 위젯, CSV 내보내기  
- 연간 29,000원 (Pro 대비 38% 할인)

### 예상 제작 기간
- MVP (체크인 + 알림 + 히스토리): 3주  
- Pro 기능 (위젯 + 리포트 + 내보내기): +3주  
- 총 6주 (혼자 개발 기준)

## 아이디어 2. FailLog

### 해결하는 문제
1인 창업자는 실패를 기록하지 않는다.  
부끄럽거나 잊고 싶어서가 아니라, 기록할 맥락이 없어서다.  
결국 같은 실수를 6개월 뒤에 반복하고, 왜 실패했는지 패턴을 모른다.  
실패 로그는 성장의 원재료인데, 아무도 체계적으로 모으지 않는다.

### 왜 모바일이어야 하는가
- 실패는 사무실 책상 앞이 아니라, 고객 미팅 직후 지하철 안에서 "아 망했다" 싶을 때 기록해야 한다.  
- 그 순간을 놓치면 영구적으로 사라진다.  
- 음성 메모 지원으로 타이핑 없이 "방금 무슨 일 있었냐면..." 한 마디로 저장 가능.  
- 잠금화면 위젯에 "이번 달 실패 횟수 N회, 배운 것 M개" 표시 — 자책이 아닌 데이터로 전환하는 시각화.

### 핵심 기능
1. 빠른 실패 캡처: 텍스트 / 음성 / 사진 중 택일, 30초 이내 저장  
2. 실패 태그 시스템: 판단착오 / 실행미스 / 정보부족 / 타이밍 등 사전 분류  
3. 3일 후 알림: "3일 전 이 실패, 지금 보면 배운 게 뭔가요?" — 냉각 후 회고 유도  
4. 패턴 대시보드: 월별 실패 유형 분포, 같은 태그 반복 경보  
5. 실패→액션 연결: 각 실패에 "다음엔 이렇게" 한 줄 액션 저장, 완료 여부 추적

### 수익 모델
- 무료: 실패 기록 무제한, 30일 히스토리, 기본 태그  
- Pro 월 4,900원: 무제한 히스토리, 패턴 대시보드, 음성 메모, 3일 후 알림 커스터마이징  
- 팀 플랜 월 9,900원/팀(5인): 팀 실패 로그 공유, 공동 액션 아이템

### 예상 제작 기간
- MVP (기록 + 태그 + 3일 후 알림): 4주  
- Pro 기능 (음성 + 패턴 대시보드 + 팀 플랜): +4주  
- 총 8주 (혼자 개발 기준)

## 아이디어 3. OnePercent

### 해결하는 문제
"매일 1% 성장"이라는 말은 많이 들었지만, 실제로 무엇을 1% 개선할지 모른다.  
1인 창업자는 동시에 마케팅·개발·운영·CS를 하면서 어느 영역이 병목인지 파악이 안 된다.  
주간 회고를 해도 "바빴다"는 감상만 남고, 어느 영역이 성장했는지 수치로 보이지 않는다.

### 왜 모바일이어야 하는가
- 성장 습관은 아침 루틴과 연결될 때 가장 강력하다. 모바일 알람과 통합된 아침 5분 루틴이 가능하다.  
- 위젯으로 "오늘 어느 영역에 시간을 썼냐"를 잠금화면에서 확인하면, 의사결정 편향을 실시간으로 인식한다.  
- 오프라인 기록 후 동기화 — 인터넷 없는 이동 중에도 기록 가능.  
- 웹 대시보드는 주 1회 열지만, 모바일 위젯은 하루 수십 번 눈에 들어온다.

### 핵심 기능
1. 영역 설정: 개발 / 마케팅 / 운영 / 학습 / 건강 등 본인이 추적할 5개 영역 설정  
2. 일일 5분 체크: 각 영역에 오늘 투자한 시간(슬라이더) + 한 줄 성과  
3. 1% 미션: 매일 아침 "오늘 딱 하나, 어디를 1% 개선할 거예요?" — AI 기반 제안(전날 데이터 참고)  
4. 영역별 성장 그래프: 주·월 단위로 어느 영역이 올라가고 어느 영역이 방치됐는지 시각화  
5. 잠금화면 위젯: 오늘 체크된 영역 수 / 연속 기록일 / 이번 주 가장 성장한 영역  
6. 주간 리포트 카드: 공유 가능한 이미지 형태(인스타·X 공유용)

### 수익 모델
- 무료: 영역 3개, 30일 히스토리, 기본 그래프  
- Pro 월 4,500원: 영역 무제한, 무제한 히스토리, AI 미션 추천, 주간 리포트 카드 생성, 잠금화면 위젯  
- 연간 35,000원 (Pro 대비 35% 할인)

### 예상 제작 기간
- MVP (영역 체크 + 그래프 + 알림): 3주  
- Pro 기능 (AI 추천 + 위젯 + 리포트 카드): +4주  
- 총 7주 (혼자 개발 기준)
