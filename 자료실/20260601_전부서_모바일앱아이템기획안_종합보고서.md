## 2. 즉시 착수 가능 아이템 (5주 이내)

### ① 뀨로그 (KkuLog) `콘텐츠팀` — 2~3주
> 인스타·티스토리 운영자를 위한 콘텐츠 발행 기록 앱
- 모바일 이유: 아이디어 즉시 오프라인 저장, 카메라 직접 접근, 발행일 알림
- 수익: 연간 구독 9,900원. "100일 챌린지" 해시태그 바이럴 설계
- **30개 중 가장 빠른 MVP**

### ② 뀨루틴 (KkuRoutine) `콘텐츠팀` — 3~4주
> 1인 창업자·개발자 하루 루틴 체크 앱
- 모바일 이유: 푸시 알림 + 잠금화면 위젯 없이 루틴 앱은 작동 안 함
- 수익: 루틴 템플릿 팩 인앱 구매 + Pro 2,900원/월
- 뀨 템플릿샵과 직접 연계 가능

### ③ KkuKit Mobile `제안서팀` — 3~4주
> 개발자 이동 중 쓰는 유틸 도구 모음 (색상 추출, JSON, base64 등)
- 수익: 일회성 4,900원 Lifetime (구독 피로 없이 전환율 높음)
- 서버 비용 없음. 방치 수익 구조.

### ④ instalink 모바일 앱 `제안서팀·자동화팀·리서치팀 공통 제안` — 4~6주
> instalink 링크인바이오 모바일 관리 + 실시간 통계 앱
- 모바일 이유: 크리에이터는 촬영 현장·이동 중에 링크 수정 필요. 홈화면 위젯으로 오늘 방문자 표시
- 수익: instalink Pro 번들 + 앱 단독 월 3,900원
- **기존 API 그대로 활용, 리스크 최소**

### ⑤ SnapStock `운영관리팀` — 5주
> 카메라로 재고 즉시 등록 + 재고 소진 푸시 알림
- 모바일 이유: 현장에서 바로 찍어서 등록, 재고 소진 알림
- 사장님 키링 사업과 타깃 일치 → 본인이 첫 번째 유저
